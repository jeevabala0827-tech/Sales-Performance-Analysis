# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview
This project analyzes sales performance using SQL and Power BI. The dashboard provides key business insights such as total sales, profit, orders, product performance, and regional analysis through interactive visualizations.

---

## 🛠️ Tools Used
- Power BI
- Power Query
- MySQL
- SQL

---

## 📂 Dataset
- Sample Superstore Dataset

---

## 📈 Dashboard Features
- 💰 Total Sales KPI
- 📈 Total Profit KPI
- 📦 Total Orders KPI
- 🛒 Total Quantity KPI
- 📊 Category-wise Sales
- 🌍 Region-wise Profit
- 📅 Monthly Sales Trend
- 🏆 Top 10 Products by Sales
- 🍩 Sales Distribution by Segment
- 🎛️ Interactive Category & Region Slicers
